Comandos Essenciais - Git/GitHub (na ordem que aparecem)

git clone "link do git" - // Clona o repositorio //

cd "nome da pasta" - // acessa a pasta desejada //

git status - // Verifica se alguma alteração no repositório //

git ./ -a/ */ - // ao ver alterar no status, prepare para commitar (atualiza local) //

git commit - m "comentar a alteração"

git push origin main - // push (empurrar as alterações para o cloud - GitHub//
clear - // limpa a tela de comandos //
